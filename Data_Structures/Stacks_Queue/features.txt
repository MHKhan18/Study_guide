## Stacks:
-----------
* one-ended linear data structure
* LIFO = Last In First Out

#Usage:
- DFS

#Complexity:
Pushing: O(1)
Popping: O(1)
Peeking: O(1)
Searching: O(n)
Size: O(1)



#)Queue:
----------
->FIFO: First In First Out

*) Complexity:
Enqueue: O(1)
Dequeue: O(1)
Peeking: O(1)
Contains:O(n)
Removal: O(n)
IsEmpty: O(1)


#)Implementations:
--------------------
1) Dynamic Array:
    - Circular array for queue
2) Linked List:
    - Keep adding to the head 
3) Stacks using queues and queues using stacks:
   * queues maintain order of elements
     while stacks reverse the order
